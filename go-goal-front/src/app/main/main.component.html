What is Go Goal?

Go Goal is a web application that allows you to set goals and track your progress. It is a simple and easy to use tool that will help you achieve any of your finest aspirations.


But that's not all, we connect you with the ones you love and cherish while you chase your dreams, so that you can be sure that whatever you accomplish... means something.

How does it work?

It's simple. All you have to do is create an account. From there, it's easy. Set any goal you wish to accomplish and we'll do all of the tracking for you.


Complete. Edit. Set. Delete. You have full control over all of your goals. View all of them at any time and anyone else's that's signed up with us. Friends, family members, the works!
